# Slack Bot

A simple interactive nodejs slack bot.


## Configure

Go to https://slack.com/apps/A0ZC8RE4T-mfyzsawesomebot, install and obtain access token and bot tokens.

Create .env file and place tokens here:

```
SLACK_ACCESS_TOKEN='...'
SLACK_BOT_TOKEN='...'
```
